# ExcalidrawAI Frontend

Frontend client for the ExcalidrawAI project.  
Built using React and the official Excalidraw library to provide an interactive whiteboard experience with AI-powered diagram generation.

Because apparently drawing boxes manually in 2026 is still considered "productivity". Humanity truly peaked at sticky notes.

---

## Features

- Interactive drawing canvas using Excalidraw
- Real-time diagram editing
- AI prompt-based diagram generation
- Clean React component architecture
- REST/WebSocket integration support
- Modular frontend structure
- TypeScript support
- Responsive UI

---

## Tech Stack

- React
- TypeScript
- Vite
- Excalidraw
- Axios
- Tailwind CSS
- WebSocket / Socket.IO (if enabled)
